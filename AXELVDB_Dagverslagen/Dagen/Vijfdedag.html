Vandaag begon mijn dag met het controlleren van de servers. Omdat de servers elke nacht afgesloten worden is het belangrijk dat ik dit doe. Bij het herstarten van de servers is er altijd de kans dat er iets misloopt. Ook vandaag was dit het geval. Toen ik ging kijken in het menu van de server zag ik direct dat de server een tekort aan memory had. Om dit probleem op te lossen en te voorkomen dat dit nog is zou gebeuren heb ik wijzingen gemaakt aan de server. Ik heb memory sticks gepakt van een server die teveel memory had en deze ingestoken bij de server die een te kort heeft aan memory. Nadat ik het probleem verholpen had, heeft mijn stagementor gezegd dat hij deze express verstoken heeft om te zien of ik het zou probleem zou kunnen opsporen en oplossen. Voor de rest van de voormiddag heb ik de servers verder gemonitord. In de namiddag heeft mijn stagebegleider gevraagd of ik een scriptje wouw schrijven in powershell voor het automatiseren van het opstarten en aflsuiten van de servers. Ondanks dat ik weinig kennis heb in het maken van scriptjes in powershell ben ik deze uitdaging niet uit de weg gelopen en heb ik mijn kans gegrepen om dit bij te leren. Ik ben direct op internet opzoek gegaan en heb zo mezelf de basis van powershell aangeleerd om dit scriptje te maken. Met de hulp van een chat ai bot, minimale kennis van powershell en logisch te redeneren is het gelukt om dit script te maken. Of het scriptje foutloos gaat werken is het wachten tot morgen en de verdere toekomst maar ik en mijn stagebegleider hebben er alvast veel vetrouwen in.

Axel Van den Bosch
